NAME

    YPCnv - Yellow Pages Converter

SYNOPSIS

    ypcnv [options]

DESCRIPTION

    When by CLI options it is specified enough information for a 
    conversion to be processed, the conversion will be started. In 
    other cases some UI will be shown.

    --from=FileFormatName
        Input file format special name. It is same as extension.

    --to=FileFormatName
        Output file format special name. It is same as extension.

    --if="file-name"
        Input file.

    --of="file-name"
        Output file.

FILES

    ypcnv.log, ypcnv-stderr.log
